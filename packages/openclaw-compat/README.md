# openclaw-compat

Temporary compatibility layer for retained OpenClaw logic.

**This package is intentionally empty in v0.1.**

## Purpose

As PnLClaw evolves from its OpenClaw origins, some patterns may need a transitional home. This package serves that role.

## Rules

- Do **not** add new core business logic here
- Gradually migrate existing patterns to first-party packages
- This package should shrink over time, not grow
